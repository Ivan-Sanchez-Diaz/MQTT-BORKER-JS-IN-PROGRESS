IOT-MQTT

MQTT broker that receives all the messages from a greenhouse and then displays them graphically while sending the data to a PostgreSQL database.

Display

To Interact

Send

(you have to have mosquitto installed, link here*) to send through mqtt you need -h which is the server, in this case we are using a free server for testing, you need -T which is the topic, -P which is the port and finally -M which is the message.

CMD

> mosquitto_pub -h test.mosquitto.org -t casa/datos -p 1883 -m "anything"

Receive

(you have to have mosquitto installed, link here*) to receive through mqtt you need -h which is the server, in this case we are using a free server for testing, you need -T which is the topic, -P which is the port.

CMD

> mosquitto_sub -h test.mosquitto.org -t casa/datos -p 1883
